# Password Manager - Backend

Безопасная система хранения паролей с использованием современных криптографических алгоритмов.

## 📁 Структура проекта

```
bez/
├── backend/
│   ├── core/           # Криптографическое ядро
│   ├── database/       # Работа с БД
│   ├── features/       # Дополнительные функции
│   ├── tests/          # Тесты
│   └── api/            # REST API
├── docs/
│   ├── guides/         # Руководства по использованию
│   ├── security/       # Документы по безопасности
│   ├── reports/        # Отчёты и changelog
│   └── ...             # Спецификации и диаграммы
├── .github/workflows/  # CI/CD pipeline
├── README.md           # Этот файл
├── CONTRIBUTING.md     # Как внести вклад
├── requirements.txt    # Зависимости Python
└── pyproject.toml      # Конфигурация проекта
```

## 🚀 Быстрый старт

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск тестов
```bash
pytest backend/tests/ -v
```

### Запуск приложения
```bash
uvicorn backend.api.main:app --reload
```

## 🔐 Криптография

| Алгоритм       | Назначение       | Параметры            |
|----------------|------------------|----------------------|
| AES-256-GCM    | Шифрование данных| 256-битный ключ      |
| Argon2id       | Деривация ключей | t=3, m=64MB, p=4     |
| HMAC-SHA256    | Целостность данных| 256-битный ключ     |
| SHA-256        | Хеширование      | 256-битный хеш       |
