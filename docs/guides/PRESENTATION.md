## Обзор проекта

### Цель

Создание безопасного менеджера паролей с использованием современных криптографических алгоритмов.

### Ключевые требования

- ✅ AES-256-GCM шифрование
- ✅ Argon2id деривация ключей
- ✅ Защита от брутфорс атак
- ✅ Audit logging всех операций
- ✅ Покрытие тестами >85%
- ✅ 0 критических уязвимостей

### Технологии

| Категория | Технологии |
|-----------|------------|
| **Язык** | Python 3.11+ |
| **Криптография** | cryptography, argon2-cffi, PyNaCl |
| **Веб-фреймворк** | FastAPI |
| **База данных** | SQLite + SQLAlchemy |
| **Валидация** | Pydantic v2 |
| **Тестирование** | pytest, pytest-cov, pytest-benchmark |
| **Безопасность** | bandit, safety, mypy |
